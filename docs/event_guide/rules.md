# Rules and Flags

1. Select **RULES & FLAGS**
2. Configure the rules and flags:

    <pre>
    Rules and Flags: **Off**
    Parc Fermé Rules: **Off**
    Pit Stop Experience: **Broadcast**
    </pre>

3. Return to the **Grand Prix™ Settings** screen
